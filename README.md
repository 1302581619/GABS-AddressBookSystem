## 项目特色
* 前后端分离，前端尽量使用纯HTML+JS，通过AJAX与后端PHP接口交互，便于维护和技术升级。
* 重要数据加密，用户登录信息、通讯录数据完全加密。
* 支持多语言，语言种类可轻松扩展。

## 部署方式
* 1、WAMP或LAMP环境
* 2、创建数据库、运行sql\db.sql文件
* 3、修改数据库连接配置文件：common\db.php
* 4、完成
